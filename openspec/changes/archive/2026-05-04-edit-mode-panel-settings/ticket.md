# HEL-176 — Edit mode: consolidate all panel settings

## Title
Edit mode: consolidate all panel settings

## Description
Edit mode surfaces all panel configuration in one place: appearance (title, colors, border), data binding (DataType selection), field mapping, and refresh interval. Replaces the current customization popover — popover can be removed once this ships.

## Priority
High

## Project
Helio v1.2 — Panel System
