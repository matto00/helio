# HEL-159: Implement CSS container queries for panel content re-flow

## Title
Implement CSS container queries for panel content re-flow

## Description
Establish the sizing system infrastructure using CSS container queries (or equivalent). Panel content should re-flow and resize correctly as the panel container dimensions change via grid drag/resize.

## Priority
High

## Project
Helio v1.1 — UX Foundations

## Parent
HEL-136

## Acceptance Criteria
Panel content should re-flow and resize correctly as the panel container dimensions change via grid drag/resize. This requires establishing the CSS container query (or equivalent) sizing system infrastructure so that panel inner content can respond to the panel's own size rather than the viewport size.
