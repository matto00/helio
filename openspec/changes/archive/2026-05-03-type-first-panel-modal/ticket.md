# HEL-168 — Type-first panel creation modal

## Title
Type-first panel creation modal

## Description
Replace the current creation form with a modal that begins with panel type selection. User picks a type before any other configuration is shown. Replaces the existing create flow end-to-end.

## Acceptance Criteria
- A modal opens when the user initiates panel creation
- The first step of the modal presents a panel type picker (all available types)
- Only after a type is selected does the user proceed to any further configuration
- The existing create panel flow is replaced entirely by this new modal
- The new modal is visually consistent with the existing Helio UI patterns

## Priority
High

## Project
Helio v1.2 — Panel System
