# HEL-169 — Panel type picker with visual cards

## Title
Panel type picker with visual cards

## Description
Visual card grid in the creation modal: one card per panel type, each showing an icon, type name, and a one-line description. Selected card is highlighted. Covers all current and cosmetic panel types.

## Priority
High

## Project
Helio v1.2 — Panel System

## Parent
HEL-138

## Acceptance Criteria
- A visual card grid is shown in the panel creation modal
- Each card displays: an icon, the panel type name, and a one-line description
- The currently selected card is visually highlighted/active
- All current panel types are represented (and any cosmetic panel types)
- Selecting a card sets that panel type as the chosen type
