# HEL-171 — Live inline preview during panel creation

## Title
Live inline preview during panel creation

## Description
Panel preview in the creation modal updates live as the user configures settings. Shows a representative render of the panel with any entered content or selected options applied.

## Acceptance Criteria
(Derived from ticket description)
- The panel creation modal includes a live preview section
- The preview updates in real-time as the user changes settings (title, content, panel type options)
- The preview shows a representative render of the panel as it would appear on the dashboard
- The preview reflects the currently selected panel type
- The preview reflects any entered content or selected options

## Context
This ticket is part of the "Helio v1.2 — Panel System" project and is a child of HEL-138.

The panel creation modal was recently updated by HEL-168 and HEL-169 to include a type-first
selection step with visual card descriptions. This ticket adds live preview functionality to
the configuration step of that modal.
