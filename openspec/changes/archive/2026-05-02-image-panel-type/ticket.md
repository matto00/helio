# HEL-165 — Image panel type

## Title
Image panel type

## Description
Static image panel via URL or file upload. Configurable image fit: contain / cover / fill. No DataType binding required. Image stored as a URL reference on the panel record.

## Acceptance Criteria
- A new panel type "image" is available when creating panels
- The image panel accepts a URL as the image source
- The image panel supports configurable image fit: contain / cover / fill
- No DataType binding is required for the image panel type
- The image URL is stored as a URL reference on the panel record
- The image panel renders the image in the dashboard grid
- The image fit setting is persisted and respected on re-load

## Priority
High
