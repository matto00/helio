# HEL-163 — Apply sizing system to table panel

## Title
Apply sizing system to table panel

## Description
Apply the sizing system to the table panel type. Table fills the panel area; rows visible without dead space. Scrollable when content exceeds panel height.

## Acceptance Criteria
- Table panel fills the full panel area (no dead space)
- All rows are visible without unnecessary vertical whitespace
- The table scrolls vertically when content exceeds the panel height
- Sizing behaviour matches the pattern established by other panel types in the sizing system
