# HEL-166: Divider panel type

## Title
Divider panel type

## Description
Horizontal or vertical rule for visual grouping. Configurable weight and color. Purely cosmetic — no content or data binding.

## Acceptance Criteria
- Panel type "divider" is available when creating a new panel
- Divider renders as a horizontal or vertical rule on the dashboard
- Orientation (horizontal/vertical) is configurable
- Weight (thickness) is configurable
- Color is configurable
- No content or data binding — purely cosmetic
- Divider persists via the existing panel API (panel type stored as "divider")
- Divider panel appearance is editable via the panel settings UI

## Context
- Parent issue: HEL-137 (Panel System)
- Project: Helio v1.2 — Panel System
- Priority: Medium
