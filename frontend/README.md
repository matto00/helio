# Frontend Scaffold

React + TypeScript + Redux frontend scaffold.

No feature implementation is included yet.

Planned structure:

- `src/app` application bootstrap and providers
- `src/components` shared presentational components
- `src/features` feature modules (`dashboards`, `panels`, etc.)
- `src/store` Redux store setup and slices
- `src/hooks` reusable hooks
- `src/services` API and client-side services
- `src/types` shared TypeScript types
- `src/utils` utility helpers
- `src/styles` global styles and design tokens
- `tests` frontend tests (Jest)
